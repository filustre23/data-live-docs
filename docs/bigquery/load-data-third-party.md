# 使用第三方應用程式載入資料

部分第三方應用程式和服務提供連接器，可將資料擷取至 BigQuery。舉例來說，[Informatica](https://docs.cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=zh-tw#informatica) 可讓您將外部來源的資料載入 BigQuery 儲存空間。

## 使用 Informatica 載入資料

如要使用 Informatica Data Loader 載入資料，請按照下列步驟操作：

1. 前往 Google Cloud 控制台的「BigQuery」頁面。

   [前往 BigQuery](https://console.cloud.google.com/bigquery?hl=zh-tw)
2. 在「Explorer」窗格中，按一下 add「新增」。

   「Add data」對話方塊隨即開啟。
3. 按一下資料來源，例如 **Amazon S3** 或 **Google Cloud AlloyDB**。
4. 按一下「Informatica Data Loader」解決方案資訊卡。
5. 如果您同意「您即將離開 Google」對話方塊中的條款，請按一下「確認」。
6. 在 Informatica 登入頁面上，登入 Informatica 帳戶。
7. 如要將資料載入 BigQuery，請按照 Informatica 帳戶中的步驟操作。

## 使用 Fivetran 載入資料

如要使用 Fivetran Data Pipelines 載入資料，請按照下列步驟操作：

1. 前往 Google Cloud 控制台的「BigQuery」頁面。

   [前往 BigQuery](https://console.cloud.google.com/bigquery?hl=zh-tw)
2. 在「Explorer」窗格中，按一下 add「新增」。

   「Add data」對話方塊隨即開啟。
3. 按一下資料來源，例如 **Amazon S3** 或 **Google Cloud AlloyDB**。
4. 按一下「Fivetran Data Pipelines」解決方案資訊卡。
5. 如果您同意「您即將離開 Google」對話方塊中的條款，請按一下「確認」。
6. 在 Fivetran 登入頁面上，登入 Fivetran 帳戶。
7. 如要將資料載入 BigQuery，請按照 Fivetran 帳戶中的步驟操作。
