# BigQuery DataFrames Python API

[BigQuery DataFrames](https://dataframes.bigquery.dev/) 是 Python API，可用於在 BigQuery 中分析資料及執行機器學習作業。

BigQuery DataFrames 包含以下部分：

* `bigframes.pandas` 會在 BigQuery 上實作 [類似 pandas 的 API](https://dataframes.bigquery.dev/reference/api/bigframes.pandas.html)。詳情請參閱「[支援的 pandas API](https://dataframes.bigquery.dev/supported_pandas_apis.html)」。
* `bigframes.ml` 會在 BigQuery ML 上實作 [scikit-learn 類似 API](https://dataframes.bigquery.dev/reference/api/bigframes.ml.html)。

如要開始使用 BigQuery DataFrames，請參閱 [BigQuery DataFrames 快速入門指南](https://docs.cloud.google.com/bigquery/docs/dataframes-quickstart?hl=zh-tw)。
