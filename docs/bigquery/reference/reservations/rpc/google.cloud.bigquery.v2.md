## LocationMetadata

BigQuery-specific metadata about a location. This will be set on google.cloud.location.Location.metadata in Cloud Location API responses.

| Fields | |
| --- | --- |
| `legacy_location_id` | `string`  The legacy BigQuery location ID, e.g. "EU" for the "europe" location. This is for any API consumers that need the legacy "US" and "EU" locations. |
