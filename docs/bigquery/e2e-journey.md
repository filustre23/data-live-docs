# 機器學習模型的端對端使用者歷程

本文說明在 BigQuery ML 中訓練的機器學習模型使用者歷程，包括可用於處理機器學習模型的陳述式和函式。BigQuery ML 提供下列類型的機器學習模型：

* [監督式學習](https://developers.google.com/machine-learning/glossary/fundamentals?hl=zh-tw#supervised-machine-learning)模型：

  + [線性迴歸和邏輯迴歸](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-glm?hl=zh-tw)
  + [深層類神經網路 (DNN)](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-dnn-models?hl=zh-tw)
  + [廣度和深度](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-wnd-models?hl=zh-tw)
  + [強化型樹狀結構](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-boosted-tree?hl=zh-tw)
  + [隨機森林](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-random-forest?hl=zh-tw)
  + [AutoML](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-automl?hl=zh-tw)
* [非監督式學習](https://developers.google.com/machine-learning/glossary/fundamentals?hl=zh-tw#unsupervised-machine-learning)模型：

  + [K-means 分群模型](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-kmeans?hl=zh-tw)
  + [矩陣分解](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-matrix-factorization?hl=zh-tw)
  + [自動編碼器](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-autoencoder?hl=zh-tw)
  + [主成分分析 (PCA)](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-pca?hl=zh-tw)
* [僅轉換](https://docs.cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-transform?hl=zh-tw)模型：僅轉換模型並非一般的機器學習模型，而是將原始資料轉換為特徵的構件。
