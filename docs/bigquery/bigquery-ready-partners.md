# Google Cloud Ready - BigQuery 合作夥伴

合作夥伴可透過 BigQuery 驗證計畫，驗證與 BigQuery 的整合，確保整合效果達到最佳狀態。我們的 BigQuery 驗證計畫分為三個階段：

* **評估：**Google 團隊會在沙箱式正式環境中執行一系列資料整合測試，並根據最佳做法和成效基準比較結果。
* **強化：**我們會在評估階段結束後，與合作夥伴攜手合作，填補任何缺口。
* **啟用：**我們與合作夥伴攜手改善現有文件，確保共同客戶擁有順利使用我們產品所需的一切資源。

本文件列出已通過 [**Google Cloud Ready - BigQuery** 方案驗證的合作夥伴解決方案](https://docs.cloud.google.com/bigquery/docs/bigquery-ready-overview?hl=zh-tw)。如果 Cloud Customer Care 判斷 BigQuery 運作正常，就不會支援第三方軟體。這些解決方案分為下列類別：

* [商業智慧、機器學習與進階數據分析](#bi-ml-analytics)
