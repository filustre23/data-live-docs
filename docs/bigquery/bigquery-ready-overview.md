# Google Cloud Ready - BigQuery

Google Cloud 「Ready - BigQuery」計畫會認可符合一系列核心要求的合作夥伴解決方案，確保合作夥伴產品與 BigQuery 完美整合。通過驗證的合作夥伴產品可協助現有和潛在客戶發掘並使用經過驗證的合作夥伴產品。BigQuery 客戶現在可以節省評估新工具的時間，專心使用合作夥伴產品建構解決方案。這些產品經過嚴格的驗證程序，證明能與 BigQuery 完美搭配運作。

## Google Cloud Ready - BigQuery 的運作方式為何？

合作夥伴可透過 BigQuery 驗證計畫，驗證與 BigQuery 的整合，確保提供最佳客戶體驗。我們的 BigQuery 驗證計畫分為三個階段：

* **評估：**Google 團隊會在沙箱式實際工作環境中執行一系列資料整合測試，並根據最佳做法和成效基準比較結果。
* **強化：**我們會在評估階段結束後，與合作夥伴攜手填補任何缺口。
* **啟用：**我們與合作夥伴攜手修訂現有文件，確保共同客戶能順利使用整合式產品實作解決方案。

## 計畫範圍和合作夥伴福利

合作夥伴註冊加入這項計畫後， Google Cloud 團隊會測試其 BigQuery 整合的品質。合作夥伴接著可以著手解決任何已發現的缺口或需要改進之處。合作夥伴整合功能通過驗證後，我們就會授予合作夥伴「***Google Cloud Ready - BigQuery***」稱號。

這項計畫會為合作夥伴提供軟體全面審查服務，確保他們遵循 BigQuery 最佳做法，並找出可改善之處，而且完全免費。

以下列出參與計畫的 ISV 可享有的福利：

1. 合作夥伴解決方案通過驗證，可與 BigQuery 搭配使用，讓客戶更有信心。
2. 合作夥伴徽章和「***Google Cloud Ready - BigQuery***」認證。
3. 優先名列 Google Cloud
   [BigQuery 合作夥伴頁面](https://docs.cloud.google.com/bigquery?hl=zh-tw#section-13)，並獲得認可。
4. 除了標準「優勢合作夥伴」計畫提供的活動外，還支援其他 GTM 活動。這些額外福利包括聯合網誌、公關、研討會、網路研討會等。
5. 與合作夥伴工程團隊和 BigQuery 產品團隊共同制定藍圖，並搶先瞭解每季發布的專屬功能。 Google Cloud
6. 支援建立最佳做法指南、解決方案簡報和聯合參考架構。

## 如何選取合作夥伴進行驗證？

如要加入這項計畫，合作夥伴必須符合下列基本條件：

* 合作夥伴產品包含生產品質的 BigQuery 整合功能。
* 至少有 5 位客戶在正式環境中使用合作夥伴產品和 BigQuery。
* 1 份公開的客戶個案研究。

視驗證資源的可用性而定， Google Cloud 合作夥伴團隊
會與您協調，安排驗證時間。
