# Episode 2: Navigating the UI

Learn to navigate the Hex UI and Notebook view.

#### In this episode[​](#in-this-episode "Direct link to In this episode")

episode 2 of our Hex Foundations video course continues with a walkthrough of the Hex UI, providing insight into its layout and how to navigate it. We introduce you to the Notebook view (fka Logic view), the core workspace of all the analysis you’ll ever do in Hex. We will guide you through adding and deleting cells, highlight the types of cells you can work with, and give you a glimpse of writing SQL code in your projects. With each step along this path, you’re inching closer to becoming a Hex master!

**Topics covered:**

* [Notebook view](/docs/explore-data/notebook-view/develop-your-notebook) (fka "Logic view")
* [Adding cells](/docs/getting-started/develop-your-notebook#develop-your-analysis)
* First look at [SQL cells](/docs/explore-data/cells/sql-cells/sql-cells-introduction)
