# Episode 4: Writing SQL code in Hex

Learn new SQL superpowers in Hex.

#### In this episode[​](#in-this-episode "Direct link to In this episode")

To code, or not to code? In the wondrous world of data, there is one coding language with nearly infinite utility: SQL. Designed to interact with structured data, SQL is the common tongue of the data professional. In episode 4 of our Hex Foundations course, we will show you how to work with SQL in Hex to query your data and build sophisticated analyses. We show you how to configure SQL cells, write queries, chain SQL cells together to make CTEs, and how Hex makes it painless to work with large datasets.
