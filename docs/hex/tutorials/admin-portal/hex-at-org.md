# Give your team a runbook for Hex

The details of how your company runs or how users in Hex should expect to work will vary team to team. Here's an example runbook with some suggested content to help guide new and existing Hex users on important internal processes and/or where & who to ask for help when they need it.

The Hex project below can be exported as a fully customizable template if you click the "Download template" button below. Then, you can create a new project in your own Hex workspace by [uploading the template](/docs/explore-data/projects/import-export) .yaml file and edit the placeholder text as needed! If you ever need a static version of the project, you can always [export the app as a PDF](/docs/share-insights/apps/export-as-pdf).
