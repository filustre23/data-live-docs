# Snapshots

Snapshots allow users to freeze an App in time for easy reference in the future.

info

* Available on the **Professional** [plan](https://hex.tech/pricing) and above.
* Users will need **Can Explore** or higher [project permissions](/docs/collaborate/sharing-and-permissions/project-sharing#project-permissions).

Anyone who views an App can capture a moment-in-time view of the App as a Snapshot.

## Create a snapshot[​](#create-a-snapshot "Direct link to Create a snapshot")

To create a Snapshot, click the three-dots icon at the top right of the published App. Click **Add**, enter a name for your Snapshot in the modal, and click **Save snapshot**.

[](/assets/medias/taking-snapshot-eca1d0cc123f5a718f4487548039ea0f.mp4)

## Share a Snapshot[​](#share-a-snapshot "Direct link to Share a Snapshot")

To share a Snapshot, you can distribute a link to the Snapshot. Users will need [access](/docs/share-insights/apps/publish-and-share-apps#share-your-app) to the published App in order to view the App Snapshot.

To get a link to a Snapshot, click the three-dots icon next to the Snapshot in the Snapshots list. Then click **Copy link**.

## Export a Snapshot as a PDF[​](#export-a-snapshot-as-a-pdf "Direct link to Export a Snapshot as a PDF")

To export a Snapshot as a PDF, make sure you are viewing the Snapshot you would like to export. Then, click the three-dots icon in the top-right corner and click **Export as PDF**.

## Return to the published App[​](#return-to-the-published-app "Direct link to Return to the published App")

If you are viewing a Snapshot, you can return to the latest published App by clicking the 'View live app' button in the banner at the top of the App Snapshot.
