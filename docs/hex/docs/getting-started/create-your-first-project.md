# Create your first project

Every Hex project is a launchpad to explore, transform, visualize, and operationalize your data.

## Add a project[​](#add-a-project "Direct link to Add a project")

### Create a project from scratch[​](#create-a-project-from-scratch "Direct link to Create a project from scratch")

The most common way to add a project to your Hex workspace is to create a new project from scratch. Simply click the **New project** button from your workspace Home.

### Import an existing project[​](#import-an-existing-project "Direct link to Import an existing project")

If you’d rather work off an existing project, there are a few ways to do so:

1. Import an existing .ipynb Jupyter notebook ([Read more](/docs/explore-data/projects/import-export)).
2. Import an existing .yaml Hex project ([Read more](/docs/explore-data/projects/import-export)).

### Copy a project from the Template Gallery[​](#copy-a-project-from-the-template-gallery "Direct link to Copy a project from the Template Gallery")

If you need a little extra inspiration, try copying a template project from our [Template Gallery](https://hex.tech/templates). Click **Copy this template** to automatically duplicate the project into your workspace.

## Name your project[​](#name-your-project "Direct link to Name your project")

Give your project a descriptive title. You can always change this later.
