# Connect your data

Connect data from your warehouse, cloud storage provider, file upload, or API.

tip

Anyone can start exploring Hex right away on our demo data connection, **[Demo] Hex Public Data**, which comes pre-installed on all new workspaces. If you don't see it, an Admin can restore the demo data connection from from **Settings > Data sources**.

## Connect your data to Hex[​](#connect-your-data-to-hex "Direct link to Connect your data to Hex")

There are several ways to start working with your own data in Hex:

1. Connect to your data warehouse or other database (read more in **[Data connections](/docs/connect-to-data/data-connections/data-connections-introduction)**).
2. Upload a CSV, JSON, or other file to your project (read more in **[Upload files](/docs/connect-to-data/upload-files)**).
3. Import data via API (read more in **[Import data via API](/docs/connect-to-data/import-data-via-api)**).

tip

You can combine data from multiple data sources in the same project.
