# Real-time collaboration

Hex supports live, concurrent editing workflows, making it easy to collaborate with your team.

info

* Users will need **Can Edit** [project permissions](/docs/collaborate/sharing-and-permissions/project-sharing#project-permissions) or higher to use multiplayer editing.

## Presence[​](#presence "Direct link to Presence")

Any active or previously active users will have their statuses displayed at the top of the project.

[](/assets/medias/presence-7bfa17f6b3ca66613357bb817f195801.mp4)

## Cell locking[​](#cell-locking "Direct link to Cell locking")

When a user is actively editing a cell, they will automatically apply a "lock", disallowing any other users from also applying edits.

Users can take over a cell from someone else, e.g., if they got up from their computer without releasing their lock.

[](/assets/medias/take-over-cell-deab2f7684084a66750ba3d90cf73465.mp4)
