# Models

Connect and manage Semantic Models in the Context Studio.

info

* Available on the Team and Enterprise [plans](https://hex.tech/pricing/).
* **Admins**, **Managers**, **Editors**, and **Explorers** can view the models tab.
* Only **Admins** and **Managers** can create, edit, and publish models.

## Semantic models in the context studio[​](#semantic-models-in-the-context-studio "Direct link to Semantic models in the context studio")

Semantic models define shared metrics, dimensions, and relationships used across Hex. When configured, agents can use these definitions to answer questions more consistently.

Context Studio shows how semantic models contribute to agent behavior and gives you another entry point into authoring and managing semantic models. To learn how to create and manage semantic models, see [Semantic modeling](/docs/connect-to-data/semantic-models/intro-to-semantic-models).
