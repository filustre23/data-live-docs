# User settings

Customize your personal Hex experience with user settings.

## Preferences[​](#preferences "Direct link to Preferences")

On the preferences panel, you can customize Hex with options specific to you.

### Behavior[​](#behavior "Direct link to Behavior")

* Enable AI inline completions in code cells.
* Allow cells to occupy the full width of your notebook.
* Customize word-wrap and line-wrap settings for different cell types.
* Toggle on [Vim key bindings](https://vim.rtorr.com/) for SQL, code (Python & R), and markdown cells. This setting applies globally to all Hex Notebook views you interact with. Note that when Vim key bindings are enabled, you'll need to use `shift + esc` to enter into [command mode](/docs/explore-data/notebook-view/keyboard-shortcuts#command-mode).

### Agent preferences[​](#agent-preferences "Direct link to Agent preferences")

* Specify whether you want the agent default reference to be all data or endorsed assets only in [Endorsed Mode](https://learn.hex.tech/docs/organize-content/statuses-categories#endorsed-mode).
* Customize how Hex agents respond across your projects with private, user-level instructions.

### Theme[​](#theme "Direct link to Theme")

* Select your preferred color theme for Hex. The default selection will match your system's theme preference.

## Notifications[​](#notifications "Direct link to Notifications")

Manage your notification preferences for different Hex interactions.

All email notifications are on by default. You can authenticate with your Slack account in order to customize notification preferences in Slack.

## API keys[​](#api-keys "Direct link to API keys")

Manage the tokens you have generated to access the Hex API. This section is only visible if your Admin has enabled [API access](/docs/administration/workspace_settings/workspace-security#enable-api-access).

For more information, view Hex's [API documentation](/docs/api-integrations/api/overview).
